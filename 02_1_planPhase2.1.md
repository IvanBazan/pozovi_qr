## Приоритет 2: Кнопка скачивания QR

Добавить под QR-кодом кнопку "Скачать PNG" (или SVG).
QR-код генерится через `qrcode.react` — можно отрендерить в canvas и вызвать `toDataURL()` для скачивания.

Пример подхода:
```jsx
import { QRCodeCanvas } from 'qrcode.react';

// ref на canvas
const qrRef = useRef();

const handleDownload = () => {
  const canvas = qrRef.current.querySelector('canvas');
  const url = canvas.toDataURL('image/png');
  const a = document.createElement('a');
  a.href = url;
  a.download = `qr-${slug}.png`;
  a.click();
};

<div ref={qrRef}>
  <QRCodeCanvas value={fullUrl} size={256} />
</div>
<button onClick={handleDownload}>Скачать PNG</button>
```

Если сейчас используется `QRCodeSVG` — переключить на `QRCodeCanvas` для скачивания, или реализовать SVG-скачивание через `new XMLSerializer().serializeToString()`.

## Приоритет 3: Мелкие улучшения (опционально)

### Название выбранной ссылки над QR
Сейчас над QR показывается только URL. Добавить title или slug крупным текстом — чтобы было понятно, к какой ссылке относится правая панель.

### Скрыть колонку "Страна" если нет данных
Пока GeoIP не подключен, все значения `—`. Можно скрыть колонку или показывать её только когда есть хотя бы одно непустое значение.

### Валидация URL в модалке
Простая клиентская проверка: `target_url` должен начинаться с `http://` или `https://`.

### Состояние "ничего не выбрано"
При первой загрузке правая панель должна показывать placeholder: "Выберите ссылку из списка".
